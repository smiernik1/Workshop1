# Tasks Management Console Application
## Opis
Ten projekt to aplikacja konsolowa do zarządzania zadaniami, stworzona w ramach warsztatów programistycznych. Aplikacja umożliwia użytkownikom dodawanie, usuwanie i przeglądanie zadań.
## Funkcjonalności

- **Dodawanie zadań**: Użytkownik może dodawać nowe zadania do listy.
- **Usuwanie zadań**: Użytkownik może usuwać istniejące zadania.
- **Wyświetlanie zadań**: Użytkownik może przeglądać listę wszystkich zadań.
- **Zapis i odczyt z pliku**: Aplikacja zapisuje zadania do pliku i odczytuje je przy uruchomieniu.
## Wymagania

- Java 11 lub nowsza
- Maven (opcjonalnie, jeśli używasz do zarządzania zależnościami)

## Instalacja

1. Sklonuj repozytorium:
   bash
   git clone https://github.com/twoje-repozytorium/tasks-management.git
2. Przejdź do katalogu projektu:
   bash
   cd tasks-management
3. Skompiluj projekt:
   bash
   javac -d bin src/*.java
   
## Uruchamianie

Aby uruchomić aplikację, wykonaj następujące polecenie:

bash
java -cp bin Main

## Użycie

Po uruchomieniu aplikacji, postępuj zgodnie z instrukcjami wyświetlanymi w konsoli, aby zarządzać swoimi zadaniami.
